# 🐉 Fourth Wing - Galeria Interativa

Projeto front-end inspirado no livro **Fourth Wing**, desenvolvido com HTML, CSS e JavaScript.

A aplicação apresenta uma galeria de personagens e elementos do universo da história. Ao clicar em cada card, um **modal interativo** exibe informações detalhadas sobre o personagem ou elemento selecionado.

## 🚀 Funcionalidades

* Galeria de personagens
* Modal interativo com descrição
* Interface inspirada em fantasia
* Layout organizado em cards
* Manipulação do DOM com JavaScript

## 🛠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
